## 代码高亮

主题支持多种编程语言的代码高亮：

### Python

```python
def hello_world():
    """一个简单的 Python 函数"""
    print("Hello, World!")
    return True

result = hello_world()
print(f"函数返回结果: {result}")
```

### JavaScript

```javascript
function greet(name) {
  return `Hello, ${name}!`;
}

console.log(greet("World"));
```
