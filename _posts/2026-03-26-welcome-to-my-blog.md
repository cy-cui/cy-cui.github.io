## 代码高亮

 Chirpy 主题支持多种编程语言的代码高亮。以下是一些示例：

### Python

```python
def hello_world():
    """一个简单的 Python 函数"""
    print("Hello, World!")
    return True

# 调用函数
result = hello_world()
print(f"函数返回结果: {result}")
```

### JavaScript

```javascript
const greeting = (name) => {
  console.log(`Hello, ${name}!`);
  return true;
};

greeting('World');
```

### Bash

```bash
#!/bin/bash
echo "Hello, World!"
ls -la
```
